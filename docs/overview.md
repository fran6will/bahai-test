# Project Overview

## Purpose
The **Bahá'í Community of Montreal Website** is a platform designed to share information about the Bahá'í Faith, community activities, and events in Montreal. It serves as a hub for both community members and the general public to learn about the Faith's principles, find local gatherings, and get in touch with the community.

## Tech Stack
The project is built using a modern web development stack:

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Internationalization**: [next-intl](https://next-intl-docs.vercel.app/)
- **UI Components**: Custom components built with React and Tailwind.

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd webapp
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  **Open the application:**
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Key Directories
- `src/app`: Main application code (pages, layouts).
- `src/components`: Reusable UI components.
- `messages`: Translation files (en.json, fr.json).
- `public`: Static assets (images, fonts).
